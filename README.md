# Omega Dev - Landing Page

Esta é a landing page oficial da **Omega Dev**, desenvolvida para apresentar projetos e a identidade da marca com uma estética moderna e interativa.

## O que é?
A **Omega Dev** é uma landing page moderna e responsiva desenvolvida para servir como uma central de projetos e apresentação de marca. O projeto apresenta uma estética inspirada no universo *cyberpunk*, contando com uma seção de destaque para exibição de portfólio através de um painel interativo (slider), uma seção institucional "Quem Somos" e um rodapé com integração para redes sociais.

## Tecnologias Aplicadas
*   **HTML5:** Estruturação semântica do conteúdo (uso de tags como `header`, `main`, `section` e `footer`).
*   **CSS3:** Estilização avançada, layout responsivo e transições para os painéis de imagem.
*   **JavaScript (Vanilla JS):** Lógica personalizada para a manipulação do DOM e controle do carrossel de projetos.
*   **Font Awesome:** Biblioteca de ícones para representação visual de redes sociais.

## Conceitos Usados
*   **Manipulação de DOM:** Uso de `querySelectorAll`, `getElementById` e `classList` para alternar estados visuais dinamicamente.
*   **Lógica de Carrossel:** Implementação de controle de índices e verificação de limites (primeira/última imagem) para navegação entre painéis.
*   **Web Semântica & Acessibilidade:** Uso de atributos `aria-label`, `role="img"` e `tabindex` para garantir que o site seja compreensível por leitores de tela e navegável via teclado.
*   **Clean Code:** Organização de funções específicas (`esconderImagens`, `mostrarImagens`) para manter o código JavaScript modular e de fácil manutenção.
*   **Design Responsivo:** Configuração de viewport e estrutura flexível para adaptação em diferentes tamanhos de tela.
